# My Simple Web Portfolio

A 3-page student website built for the Web Technologies mini-project.

## Pages
- Home
- About
- Contact

## Features
- Semantic HTML structure
- External CSS for layout and design
- JavaScript menu toggle
- Dynamic date and greeting
- Contact form validation
- Responsive design
- SEO basics (title, meta description, alt text)

## Suggested Git steps
```bash
git init
git add .
git commit -m "Initial project setup"
git checkout -b feature-contact-form
git add .
git commit -m "Add contact form validation"
git checkout main
git merge feature-contact-form
```

## Deployment
Deploy this project to GitHub Pages, Netlify, or Vercel, then replace the live URL in the contact page.
