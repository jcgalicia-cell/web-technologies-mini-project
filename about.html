About the project

Designed as a simple student website with a clear structure.

The goal is to show what we learned from Modules 1 to 7 in one short, practical website.

Project idea

This website is a personal-style portfolio that introduces the project, explains the learning process, and provides a contact form. It uses straightforward layout choices so the content is easy to read and present during submission.

What we learned

  • Module 1: websites, hosting, pages, and navigation
  • Module 2: semantic HTML structure
  • Module 3: CSS styling and layout
  • Module 4: JavaScript basics
  • Module 5: responsive design
  • Module 6: Git branches and commits
  • Module 7: server and deployment basics
